# PokeAPI NestJS Clone

## Execute in development mode
1. Clone the repository
2. Execute ``` yarn install```
3. Install nest CLI ``` npm i -g @nestjs/cli ```
4. Configure DB ```docker-compose up -d```

## Tech Stach
- MongoDB
- Mongoose (Mongo DB modeling tool)
- NestJS
